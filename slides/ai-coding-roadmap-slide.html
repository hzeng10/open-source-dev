从战略判断到落地：三大转变的 AI-Coding 实施路径（3–5 年规划）

围绕 知识资本化 · 流程 Agent 化 · 交付资产化 三条主线，分阶段把战略转变沉淀为可持续的产能与差异化。

01
存量定位
成本中心 → 产能杠杆
1
建代码知识底座/图谱：统一索引几十个仓库，建依赖与调用链图谱、自动生成架构与业务规则文档，沉淀持久上下文。
2
立「AI 可读性」工程标准：结构化接口契约 + 知识卡片，让存量持续资产化、可被任意 Agent 调用。
3
知识底座服务化：作为内部知识中枢，新人与 Agent 数小时读懂存量，跨域协同有统一事实源。
🔭 3–5 年前沿
仓库级知识图谱驱动的自动定位与修复成熟，「代码知识资本」成为核心数字资产。
02
交付模式
人力堆叠 → AI 产能（人效跃升）
1
流程重构「Spec 驱动 + Agent 执行 + 人工把关」：人定义意图与验收，Agent 自主实现-自测，人专注审查与架构。
2
建 Agent 编队 + 平台编排：多 Agent 并行跨仓库作业（对齐「IDE 可选、控制权转向平台」趋势）。
3
角色/考核转型 + 治理门禁 + FinOps 看板：从「代码产量」转向「Spec 清晰度·审查效能·交付吞吐」，可控可量化。
🔭 3–5 年前沿
小型任务专用模型 × Agent 编队成主力，「小核心团队 + 大规模 Agent 产能」，工程师即编排者/审查官。
03
现场定制
一次性成本 → 可复制差异化
1
平台化 + 内核/定制分离：把定制从「改主干」变为「生成适配/插件层」，遏制分支爆炸。
2
建现场 AI 交付助手：基于知识底座，现场自然语言完成定制/配置/修复，需求 → AI 生成定制层。
3
定制资产化为模板库/行业蓝图：跨客户复制，「定制即资产」飞轮 + 自动化回归保可靠。
🔭 3–5 年前沿
现场即时定制（real-time / on-site）成护城河，从「一客一版」走向「一平台 N 蓝图」。
第 1 年夯基
建知识底座、Spec 驱动试点、平台化/内核定制分离起步（选 2–3 仓库/特性试点）。
›
第 2–3 年提效
Agent 编队规模化、现场 AI 助手上线、治理门禁 + FinOps 固化。
›
第 3–5 年领先
知识资本沉淀、模板库/行业蓝图成熟、AI 原生交付规模复制。
示意性北极星目标（目标·非承诺）： ×3–5人效 ↓50%+现场交付周期 ≥60%定制复用率 ≥70%AI 参与代码占比
